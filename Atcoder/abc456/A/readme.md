---

## 1. 문제 요약
- 문제 핵심:
  - 정수 $X$가 주사위를 3개 굴려 만든 합인지 구하시오.
- 입력/출력 조건:
- 제한 조건: $1\leq X \leq 20$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 3개의 주사위로 나올수 있는 합을 미리 구하고 $X$가 포함되는지 체크
- 막혔던 포인트:
  - x
- 답지 참고 후:
  - x

---

## 3. 코드 정리
[answer.py](./answer.py)
```py
print("NYoe s"[int(input())in{i+j+k for i in range(1,7)for j in range(1,7)for k in range(1,7)}::2])
```

---

## 4. 시간/공간 복잡도

* 시간: $O(1)$
* 공간: $O(1)$
