## 1. 문제 요약
- 문제 핵심: 
  - 면이 6개인 주사위가 3개가 있고 각각의 면에는 구간 $[1,6]$에 속하는 정수가 적혀있다.
  - 각 면이 나올 확률이 $\frac{1}{6}$ 일때 주사위 3개를 굴려 4,5,6이 각각 한번씩 나올 확률을 구하시오.
- 입력/출력 조건: 
- 제한 조건: 

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 4,5,6이 한번씩 등장할 수 있는 주사위의 배치를 모두고려후 수학을 이용해 확률을 계산후 출력한다.
- 막혔던 포인트:
  - x
- 답지 참고 후:
  - x

---

## 3. 코드 정리
[answer.py](./answer.py)
```py
d1=[*map(int,input().split())]
d2=[*map(int,input().split())]
d3=[*map(int,input().split())]
ans=0
for x,y,z in ((4,5,6),(4,6,5),(5,4,6),(5,6,4),(6,4,5),(6,5,4)):
    t=1
    t*=d1.count(x)/6
    t*=d2.count(y)/6
    t*=d3.count(z)/6
    ans+=t
print(ans)
```
