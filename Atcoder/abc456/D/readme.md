# 문제 제목: [Not Adjacent 2](https://atcoder.jp/contests/abc456/tasks/abc456_d)
- **플랫폼**: 앳코더
- **난이도**: 6/10
- **풀이 유형**: DP / 문자열

---

## 1. 문제 요약
- 문제 핵심:
  - `a,b,c`로 이루어진 문자열 $S$가 주어진다.
  - 인접한 두 문자가 서로 같지 않은 길이 1 이상의 부분 수열 개수를 구한다.
- 입력/출력 조건: 부분 수열의 개수를 998244353으로 나눈 나머지를 출력한다.
- 제한 조건: $1\leq |S| \leq 3\times 10^5$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 연속된 문자열을 압축해서 관찰시도
- 막혔던 포인트:
  - C번과 비슷하게 관찰하고 풀이하려고 시도하였음.
- 답지 참고 후:
  - 마지막에 무슨 문자를 선택했는가를 관찰하면 dp로 풀어낼 수 있다.
  - 마지막에 `a` 라는 문자를 선택했으면 `a`가 아닌 문자를 선택하거나 이것을 부분 수열의 처음으로 시작하게 하면 된다. 또는 다음 문자를 선택 안하거나
  - dp 점화식: dp[i][last] = dp[i-1][last] + $\sum$ (dp[i-1][ohter]) + 1

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py
MOD=998244353
a=b=c=0
for x in input():
    if x=='a':a+=(b+c+1)%MOD
    elif x=='b':b+=(a+c+1)%MOD
    else:c+=(a+b+1)%MOD
print((a+b+c)%MOD)
```

---

## 4. 시간/공간 복잡도

* 시간: $O(|S|)$
* 공간: $O(1)$

---

## 5. 배운 점 / 실수

* 부분 수열문제는 DP와 관련이 깊다는 것을 알게되었다.
