# 문제 제목: [Not Adjacent](https://atcoder.jp/contests/abc456/tasks/abc456_c)
- **플랫폼**: 앳코더
- **난이도**: 3/10
- **풀이 유형**: 문자열 / 구현

---

## 1. 문제 요약
- 문제 핵심:
  - `a,b,c`로 이루어진 문자열 $S$가 주어진다.
  - 인접한 두 문자가 서로 같지 않은 길이 1 이상의 부분 문자열 개수를 구한다.
- 입력/출력 조건: 부분 문자열의 개수를 998244353으로 나눈 나머지를 출력한다.
- 제한 조건: $1\leq |S| \leq 3\times 10^5$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 조건을 만족하는 부분 문자열은 "인접한 문자가 모두 다른 최대 연속 구간" 내부에서만 만들 수 있다.
  - 따라서 문자열을 왼쪽부터 탐색하면서,
같은 문자가 연속해서 등장하는 순간 현재 구간을 종료한다.
  - 길이가 N인 구간에서 만들 수 있는 부분 문자열의 개수는 $\frac{N(N+1)}{2}$이므로 각 구간마다 이를 더해주면 된다.
- 막혔던 포인트:
  - x
- 답지 참고 후:
  - x

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py
MOD=998244353
s=input()
ans=temp=0
for i in range(1,len(s)):
    if s[i]==s[i-1]:
        n=i-temp
        ans=(ans+((n+1)*n)//2)%MOD
        temp=i
n=len(s)-temp
print((ans+((n+1)*n)//2)%MOD)
```

---

## 4. 시간/공간 복잡도

* 시간: $O(|S|)$
* 공간: $O(1)$
