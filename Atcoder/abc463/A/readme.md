# 문제 제목: [16:9](https://atcoder.jp/contests/abc463/tasks/abc463_a)
- **플랫폼**: 앳코더
- **난이도**: 1/10
- **풀이 유형**: 구현 / 수

---

## 1. 문제 요약
- 문제 핵심: (짧게 2~3문장)
  - $X:Y$가 $16:9$인지 판별하시오.
  - 2
- 입력/출력 조건: 
- 제한 조건: $1\le X,Y \le 1000$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - $9X = 16Y$이라면 `Yes`아니라면 `No`라고 생각했다.
- 막혔던 포인트:
  - x
- 답지 참고 후:
  - x

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py
def main():
    input=open(0).readline
    x,y=map(int,input().split())
    print("YNeos"[y*16!=x*9::2])
    


main()
```

---

## 4. 시간/공간 복잡도

* 시간: $O(1)$
* 공간: $O(1)$
