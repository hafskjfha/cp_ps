# 문제 제목: [Array](https://atcoder.jp/contests/abc457/tasks/abc457_a)
- **플랫폼**: 앳코더
- **난이도**: 1/10
- **풀이 유형**: 구현

---

## 1. 문제 요약
- 문제 핵심: (짧게 2~3문장)
  - 길이가 $N$인 배열 $A_1,A_2,\cdots,A_N$ 이 주여졌을때 $A_X$의 값을 구하시오
- 입력/출력 조건:
- 제한 조건: $1\le X\le N \le 100$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 그냥 A[x-1]을 출력하면 된다고 생각했다.
- 막혔던 포인트:
  - x
- 답지 참고 후:
  - x

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py
def main():
    input=open(0).readline
    input()
    a=[*map(int,input().split())]
    x=int(input())
    print(a[x-1])
    

main()
```

---

## 4. 시간/공간 복잡도

* 시간: $O(1)$
* 공간: $O(N)$
