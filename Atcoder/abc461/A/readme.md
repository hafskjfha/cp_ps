- **플랫폼**: 앳코더
- **난이도**: 1/10
- **풀이 유형**: 구현

---

## 1. 문제 요약
- 문제 핵심:
  - 방어력이 $D$인 갑옷이 있다. 공격력이 $A$이하면 공격을 막아낼수 있다.
  - $A$와 $D$가 주어졌을때 갑옷은 공격을 막아낼 수 있을까?
- 입력/출력 조건:
- 제한 조건: $1\le A,D \le 100$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - $A \le D$ 인지 확인하고 `Yes`, `No`를 출력하면 되었다.
- 막혔던 포인트:
  - x
- 답지 참고 후:
  - x

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py
def main():
    input=open(0).readline
    a,d=map(int,input().split())
    print("YNeos"[a>d::2])
    
main()
```

---

## 4. 시간/공간 복잡도

* 시간: $O(1)$
* 공간: $O(1)$
