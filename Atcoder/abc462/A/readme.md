# 문제 제목: [Secret Numbers](https://atcoder.jp/contests/abc462/tasks/abc462_a)
- **플랫폼**: 앳코더
- **난이도**: 1/10
- **풀이 유형**: 구현

---

## 1. 문제 요약
- 문제 핵심: 
  - 문자열 $S$에서 숫자만 추출하여 원래 순서대로 배열한 문자열을 구하시오
  - 2
- 입력/출력 조건: 
- 제한 조건: $1\le |S|\le 50$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 문자열을 돌면서 숫자이면 출력하면 됨
- 막혔던 포인트:
  - x
- 답지 참고 후:
  - x

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py
def main():
    input=open(0).readline
    s=input().strip()
    for c in s:
        if '0'<=c<='9':
            print(c,end="")

main()
```

---

## 4. 시간/공간 복잡도

* 시간: $O(|S|)$
* 공간: $O(|S|)$
