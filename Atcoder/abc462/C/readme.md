# 문제 제목: [Not Covered Points](https://atcoder.jp/contests/abc462/tasks/abc462_c)
- **플랫폼**: 앳코더
- **난이도**: 3/10
- **풀이 유형**: 구현 / 정렬

---

## 1. 문제 요약
- 문제 핵심: 
  - 2차원 공간위에 점N개$(X_i,Y_i)$가 놓여있다. 여기서 $X,Y$는 $(1,2,\cdots N)$의 순열임이 보장된다.
  - 각 $i$에 대해 왼쪽아래 꼭짓점이 $(0,0)$이고 오른쪽위 꼮짓점이 $(X_i,Y_i)$인 변이 $x,y$축에 대해 평행한 직사각형에 대해 내부에 1번부터 $N$번 까지의 점이 모두 포함되지 않는 점의 개수는 몇개인지 구하시오.
- 입력/출력 조건: 
- 제한 조건: $1\le N \le 2 \times 10^5, 1\le X,Y \le N$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 한점이 직사각형을 구성할때 다른점이 있는지 판정하는 방법을 생각해보았으나 $O(N^2)$의 방법밖에 떠오르지 않았음
- 막혔던 포인트:
  - $X,Y$가 순열이라는 조건에 생각해보지 못해 최적화 아이디어를 찾지 못함
- 답지 참고 후:
  - 점들을 $x$에 대하여 정렬한 후에 순회한다. 순회하면서 이전까지 나온 $y$값중 가장 작은 값을 저장하면서 순회한다. 만약 $Y$가 이전에 나왔던 $y$의 최솟값보다 크지않다면 그점은 아무점도 포함하지 않는 것이다.

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py
def main():
    input=open(0).readline
    n=int(input())
    a=sorted([*map(int,input().split())]for _ in range(n))
    
    ans=0
    
    my=float('inf')
    for x,y in a:
        if y<=my:ans+=1
        my=min(my,y)
        
    print(ans)
    

main()
```

---

## 4. 시간/공간 복잡도

* 시간: $O(N\log N)$
* 공간: $O(N)$

---

## 5. 배운 점 / 실수

* 주어진 조건을 간과하지 않도록 노력해야겠다.
* 두개의 값이 변할때 한쪽을 정렬하거나 하여 고정시키는 아이디어가 있는지 생각해봐야겠다.
