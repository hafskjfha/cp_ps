- **플랫폼**: 앳코더

---

## 1. 문제 요약
- 문제 핵심: 
  - 사람 $i$는 $S_i$시간에 저택에 들어갔고 $T_i$시간에 나갔다. 그외에 시간에는 들어가거나 나가지 않았다.
  - 저택에서 살인사건이 일어났다. 범인은 2명이고 범행은 $x$시간에 일어나 $D$시간동안 일어났다. 즉 $x,x+D$시간에 끝났다.
  - $N$명의 용의자가 주어졌을때 <두 명의 범인, 범행 시작 시간 $x$>의 조합의 개수를 구하시오.
- 입력/출력 조건: (핵심만)
- 제한 조건: $2\le N \le 2\times 10^5, 1\le S_i \le T_i \le 10^6, 1\le D \le 10^6$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - $N$명의 사람들이 있었던 시간에 대해 몇명 있었는지 표시하여 구간에 대해 조합을 세면 될 것 같았다.
- 막혔던 포인트:
  - 사람들이 있었던 시간을 가진 배열에 대한 최적화 방법을 생각하지 못하였다.
  - 또한 $D$시간 겹치는지 판정하는 여부를 어떻게 짜야하는지 막혔다.
- 답지 참고 후:
  - 차분 배열을 이용하여 특정 시간대에 몇명이 있었는지 저장한다.
  - 다만 저장할때 머물었던 시간이 d시간보다 적으면 저장하지 않고, 나가는 시간을 t-d로 변형하여 정답을 탐색할때 x시간에 범행이 이루어졌을 때 사람수로 간주할수 있도록 한다.

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py

def main():
    input=open(0).readline
    n,d=map(int,input().split())
    m=10**6+1
    h=[0]*m
    
    for _ in range(n):
        s,t=map(int,input().split())
        if t-s>=d:
            h[s]+=1
            h[t-d+1]-=1
    
    cur=ans=0
    for i in range(m):
        cur+=h[i]
        ans+=cur*(cur-1)//2
    print(ans)


main()
```

---

## 4. 시간/공간 복잡도

* 시간: $O(N+M)$
* 공간: $O(M)$
