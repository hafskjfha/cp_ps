# 문제 제목: [16946 벽 부수고 이동하기 4](https://www.acmicpc.net/problem/16946)
- **플랫폼**: 백준
- **난이도**: 2/10
- **풀이 유형**: BFS / 자료구조

---

## 1. 문제 요약
- 문제 핵심:
  - $N\times M$격자판에서 0은 이동가능한 곳, 1은 벽이다.
  - 모든 벽에 대해서 벽을 부수고 이동할 수 있는 곳으로 변경한후 그 위치에서 이동할 수 있는 칸의 개수를 구하라.
- 입력/출력 조건: 모든 벽을 하나씩 부쉈을 때, 그 위치에서 이동할 수 있는 칸의 개수를 구한다.
원래 빈 칸은 0으로 출력하고, 벽이었던 칸은 구한 개수를 10으로 나눈 나머지를 출력한다.
- 제한 조건: $1\leq N,M\leq 1000$
