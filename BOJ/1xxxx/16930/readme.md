## 1. 문제 요약
- 문제 핵심:
  - $N\times M$ 격자판에서 $(x_1,y_1)$에서 $(x_2,y_2)$로 이동을 하려고 한다.
  - 상하좌우로 이동을 하되, 단위시간당 최대 $K$칸 만큼 갈수 있다.
- 입력/출력 조건: 목적지로 도착하는 최소 시간을 출력한다. 도착할 수 없다면 -1을 출력한다.
- 제한 조건: $1\leq N,M,K \leq 1000$

---

## 2. 접근 아이디어

  - [34513 룩의 이동] 문제와 아이디어가 같았다. 한칸은 최대 4번을 방문을 허용하게 하면 된다.
