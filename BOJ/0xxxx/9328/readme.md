## 1. 문제 요약
- 문제 핵심:
  - 빌딩은 가장자리의 벽이 아닌 곳을 통해 들어갈수 있다.
  - 열쇠가 필요한 문은 해당 열쇠가 있어야만 들어갈 수 있다.
  - 방의 일부 열쇠는 빌딩에 있다.
- 입력/출력 조건: 빌딩에 들어가서 얻을수 있는 문서($)의 최대 개수를 출력합니다.
- 제한 조건: $1\leq T\leq 100, 1\leq N,M \leq 100$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 문인데 열쇠가 없다면 좌표를 해시맵에 기록해둔다.
  - 새로운 열쇠를 발견하면 기존에 못 열었던 문의 좌표들을 해시맵에서 꺼내 큐에 삽입한다.
  - 위 두조건에 맞게 시작큐를 만든후 bfs를 돌린다.
  - bfs를 돌릴때도 두 조건에 맞게 처리한다.
