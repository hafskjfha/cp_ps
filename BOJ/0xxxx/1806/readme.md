# 문제 제목: [1806 부분합](https://www.acmicpc.net/problem/1806)
- **플랫폼**: 백준
- **난이도**: 5/10
- **풀이 유형**: 투포인터 / 누적합

---

## 1. 문제 요약
- 문제 핵심: 자연수로 구성된 수열에서 연속된 수들의 부분합 중에 그 합이 $S$ 이상이 되는 것 중, 가장 짧은 것의 길이를 구해야 합니다.
- 입력/출력 조건: 그러한 부분합이 없다면 0을 출력합니다.
- 제한 조건: $10\leq N \leq 100,000, 0<S\leq 10^8$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 일반적인 수열에서는 단조성이 안보이고 $A[i:j]$를 빠르게 구하기 위해 누적합을 이용해 전처리합니다.
  - 초기포인터를 0,0으로 두고 다음을 진행합니다.
    - $\text{perfix\_sum}[j]-\text{perfix\_sum}[i]<S$라면 $j$포인터를 증가시킵니다.
    - 아니라면 $\text{ans}=\min(j-i,\text{ans})$로 갱신하고 $i$포인터를 증가시킵니다.
