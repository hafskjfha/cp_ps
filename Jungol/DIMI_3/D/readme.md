# 문제 제목: [특별실 날짜표](https://jungol.co.kr/problem/8662)
- **플랫폼**: Jungol
- **난이도**: 3.7/10
- **풀이 유형**: dp

---

## 1. 문제 요약
- 문제 핵심: 
  - 길이가 $N$인 문자열 $S$에는 `O`, `X`, `?`가 있다. `O`가 연속 두번 나오거나 `X`가 연속 세번 나오는 것은 인된다.
  - 조건에 맞게 하여 `?`를 `O`또는 `X`로 바꿀때 `O`의 등장횟수의 최댓값을 구하시오.
- 입력/출력 조건: 조건을 만족하게 하여 문자열을 구성할 수 없다면 -1을 출력
- 제한 조건: $1\le N \le 10^6$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - `O`를 최대한 많이 만들어야 하므로 그리디라고 생각했다. 따라서 앞에서 부터 순회하며 `?`를 조건에 맞게 `O`로 바꾸는 그리디를 작성하였다.
- 막혔던 포인트:
  - WA를 받아 그리디가 깨지는 곳이 있다는 점을 알게되었지만 정해를 생각하지 못하였다.
- 답지 참고 후:
  - 이전 $k$의 문자들을 보고 상태가 결정 되는 것이므로 dp를 생각할수 있다.
  - dp[n][0]=n일까지 보고 마지막 문자가 O인 경우일때 최대 O의 개수, dp[n][1]=n일까지 보고 마지막 문자가 X인 경우일때 최대 O의 개수, dp[n][2]=n일까지 보고 마지막 두 문자가 XX인 경우일때 최대 O의 개수라고 상태정의를 한다.
  - 만약 `O`를 추가하려면 dp[n-1]에서 [1]또는 [2]를 선택해야한다. 다만 이전 문자가 `O`가 아니여야 한다. 따라서 dp[n][0]=max(dp[n-1][1],dp[n-1][2])+1로 전이할 수 있다.
  - 만약 `X`를 추가하려면 dp[n-1]에서 [0]을 선택하여(`O`뒤에 `X`붙이기)dp[n][1]=dp[n-1][0], [1]을 선택(`X`뒤에 `X`)하여 dp[n][2]=dp[n-1][1]로 전이 할수 있다.
  - `?`이라면 두가지 경우를 dp로 전이시킨다.
  - dp테이블을 모두 -INF로 초기화 시킨후 s[0]가 `O`라면 dp[0][0]=1로, `X`라면 dp[0][1]=0으로, `?`라면 dp[0][0]=1,dp[0][1]=0으로 설정한후 전이를 시작한다.
  - 전이가 직전 날짜 상태만 사용하므로 슬라이딩 윈도를 쓰면 공간복잡도는 O(1)로 줄일 수 있다.

## 3. 코드 정리
[정답 코드](./answer.py)
```py
import sys

def main():
    n=int(input())
    s=input().strip()
    
    if "OO" in s or "XXX" in s: return print(-1)
    
    INF=float('-inf')
    dp=[[INF]*3 for _ in range(n)]
    
    if s[0]=='O':dp[0][0]=1
    elif s[0]=='X':dp[0][1]=0
    else:
        dp[0][0]=1
        dp[0][1]=0
    
    for i in range(1,n):
        c=s[i]
        if c=='O':
            dp[i][0]=max(dp[i-1][1],dp[i-1][2])+1
        elif c=='X':
            dp[i][1]=dp[i-1][0]
            dp[i][2]=dp[i-1][1]
        else:
            dp[i][0]=max(dp[i-1][1],dp[i-1][2])+1
            dp[i][1]=dp[i-1][0]
            dp[i][2]=dp[i-1][1]
    
    ans=max(dp[n-1])
    print(-1 if ans==INF else ans)

    
main()
```

---

## 4. 시간/공간 복잡도

* 시간: $O(N)$
* 공간: $O(N)$

---

## 5. 배운 점 / 실수

* 마지막 $k$개의 상태만 중요하다는 DP로 생각해볼수 있다는 것을 알게되었다.
* 문자열에서 특정 패턴을 금지시키면서 무언가를 치환해 최대화하는 것을 구해야 한다면 DP를 생각해봐야 한다는 점을 알게되었다.
* 불가능한 상태를 -INF로 두고, 허용되는 전이만 작성하면 금지 패턴은 자연스럽게 배제된다.
