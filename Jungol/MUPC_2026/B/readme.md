## 1. 문제 요약
- 문제 핵심: 
  - 칠판에 $X=0$이 적혀 있고 양의 숫자가 적혀 있는 카드 $N$개가 놓여 있다. $A$와 $B$는 이것들을 이용해 대결을 하려고 한다.
  - 각 턴에는 남아 있는 카드중 하나를 반드시 선택해서 사용해야 하며 사용한 카드는 즉시 버린다.
  - 카드를 사용할 경우, 현재 칠판에 적혀 있는 값 $X$와 선택한 카드에 쓰여 있는 수 $c_i$에 대해서 $\land, \lor, \oplus (\text{xor})$ 3개 중 하나의 비트 연산을 적용할 수 있다. 적용한 결과의 값을 $Y$라고 할 때, 칠판에 적혀 있던 $X$를 지우고 대신 $Y$를 적는다.
  - 모든 카드를 사용했을 때 $A$는 칠판에 홀수가 적혀 있도록, $B$는 짝수가 적혀있으면 이긴다. $A$부터 게임이 시작했을때 누가 이기는 지 구해보자.
- 입력/출력 조건: $A$가 이기면 `myongjin`, $B$가 이기면 `hambak`출력
- 제한 조건: $1\le N \le 10^6, 1\le c_i \le 2^{20}$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - $A$는 홀수를 만들어야 한다. 짝수가 적혀있을때는 홀수를 or연산하면 홀수를 만들수 있고 홀수가 적혀있을떄는 짝수랑 or을 하거나 홀수랑 and연산을 하여 홀수를 만들 수 있다.
  - $B$는 짝수를 먼둘어야 한다. 짝수가 적혀있을때는 짝수와 or하거나 홀수와 and로 짝수를 만들수 있고 홀수가 적혀있을때는 홀수와 xor하거나 짝수와 and하면 짝수를 만들수 있다.
  - $A$는 짝수가 칠판에 적혀있을때는 홀수만을 사용해서 홀수를 만들수 있으므로 패널티다. 따라서 $B$는 홀수를 빠르게 소진시켜 짝수만 남게 한다면 이길수 있다. 따라서 홀수의 개수가 짝수의 개수보다 같거나 작다면 선공인 $A$가 진다.
  - 만약 홀수의 개수가 짝수의 개수보다 많다면 $A$가 짝수롤 소진시켜 홀수만 남게만든다. 이후로는 $A$는 홀수, $B$는 짝수를 만들어내기만 반복하면 되므로 남은 홀수의 개수가 홀수라면 $A$가 이기고 짝수라면 $B$가 이긴다. 
- 막혔던 포인트:
  - x
- 답지 참고 후:
  - x

## 3. 코드 정리
[정답 코드](./answer.py)
```py
import sys

def main():
    input=sys.stdin.readline
    n=int(input())
    even=odd=0
    
    for x in map(int,input().split()):
        if x&1:odd+=1
        else:even+=1
    
    if even>=odd: print("hambak")
    else:
        if (odd-even)&1==1:print("myongjin")
        else:print("hambak")
    
main()
```
