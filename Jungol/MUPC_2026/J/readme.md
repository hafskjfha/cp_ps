---

## 1. 문제 요약
- 문제 핵심: 
  - $N$마리의 재규어가 있고 $i$번째 재규어는 현재 체온이 $c_i$이다.
  - 관리사는 매일 낮에 재규어 한마리를 시원한 그늘로 옮긴다. 밤이 되면 그늘로 간 재규어는 체온이 1도 감소하고 나머지 재규어들은 체온이 1도 증가한다. 이후 그늘에 이동시켰던 재규어를 원래자리로 옮긴다. 체온은 음수가 될 수 있다.
  - 모든 재규어의 체온을 같게 만들기 위해 필요한 최소 날짜 수를 구하시오.
- 입력/출력 조건: 불가능하다면 -1출
- 제한 조건: $1\le N \le 2\times 10^5, 1\le c_i \le 10^9$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 시뮬레이션을 돌리는 것을 생각했지만 $O(N\max(c_i))$ 제한이 너무커 최적화를 생각했다.
  - BOJ 21773 가희와 프로세스를 생각해보니 값을 전체다 변화시키는게 아닌 상대적으로 관리하는 점을 알아내고, 그늘로 이동시켜면 상대적으로 체온이 2도 감소하는 행위 이므로 그렇게 시뮬레이션을 돌린다면 $O(\max(c_i))$라고 생각했다. 하지만 여전히 시간복잡도가 컸다.
  - 2도씩 체온이 감소하는 것이라면 결국 체온의 홀짝성은 유지되므로 홀짝성이 다른 체온을 가진 재규어들이 있다면 불가능하다. 만약 다 같은 홀짝성을 가진 체온이라면 체온의 최솟값 $m$을 만들기 위해서 각 재규어들이 몇일을 지내야 하는지 구하는 것이 되므로 $\sum_{i=1}^{N} (\lfloor \frac{c_i-m}{2} \rfloor)$이 답이 된다.
- 막혔던 포인트:
  - x
- 답지 참고 후:
  - x

## 3. 코드 정리
[정답 코드](./answer.py)
```py
def main():
    input=open(0).readline
    
    n=int(input())
    jaguars=[*map(int,input().split())]
    
    mt=min(jaguars)
    
    ans=0
    for x in jaguars:
        if (x-mt)%2==1:
            return print(-1)
        ans+=(x-mt)//2
    print(ans)


main()
```

---

## 4. 시간/공간 복잡도

* 시간: $O(N)$
* 공간: $O(N)$
