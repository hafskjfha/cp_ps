# 문제 제목: [덜 개성있는 구간](https://jungol.co.kr/problem/13632)
- **플랫폼**: Jungol
- **난이도**: 3.1/10
- **풀이 유형**: 해구성

---

## 1. 문제 요약
- 문제 핵심: 
  - 다음 조건을 만족하는 길이 $N$의 수열 $A$를 구하시오.
    - 수열 $A$의 모든 원소는 서로 다르다.
    - $1 \le L \le R \le N$을 만족하는 모든 정수 쌍 $(L, R)$에 대하여, $\sum\limits_{i=L}^R A_i$가 가질 수 있는 서로 다른 값의 개수는 정확히 $N$개이다.
- 입력/출력 조건: 조건을 만족하는 수열이 여러개라면 아무거나 출력한다.
- 제한 조건: $1\le N \le 5000,$ 수열의 구성하는 원소 $A_i$는 $-10^9 \le A_i \le 10^9$을 만족해야 한다.

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 길이가 1,2,3일때 구성되는 규칙을 찾고 4이상에 적용방법을 생각해내려고 했다.
  - 길이가 2일떄는 부분합이 $A_1, A_2, A_1+A_2$가 있고 이게 집합으로 나타냈을때 크기가 2일려면 한개가 0이면 될 것 같다고 생각했다.
- 막혔던 포인트:
  - 4이상일때 일반화 시키기에 실패하였다.
- 답지 참고 후:
  - $N=2$일때 $A_1,A_2,A_1+A_2$의 부분합이 생기는데 $A_1 \neq A_2$이니까 $A_1+A_2$가 $A_1$이거나 $A_2$이어야 한다. 따라서 두 원소중 하나는 0이고 수열을 $[0,k]$로 나타낼수 있다.
  - $N=3$일때는 $[0,k,l]$가 되고 부분합은 $l$과 $k+l$이 추가되고 $k+l=k$이면 $l=0$이라서 안되고 $k+l=0$이면 $l=-k$이므로 수열은 $[0,k,-k]$
  - $N=4$이면 $[0,k,-k,m]$이 되고 새롭게 생기는 부분합은 $m,m-k$이고 $m-k=0$이면 $m=k$이니까 불가능, $m-k=k$이면 $m=2k$, $m-k=-k$이면 $m=0$이니까 불가능, $m-k=m$이면 $-k=0$이니까 불가능 따라서 $[0,k,-k,2k]$이 되고 $N>4$이상에서도 $[0,k,-k,2k,-2k,\dots]$ 이와 같은 규칙을 가진다는 것을 알수 있다.

## 3. 코드 정리
[정답 코드](./answer.py)
```py
import sys

def main():
    input=sys.stdin.readline
    n=int(input())-1
    a=[0]
    
    for i in range(n):
        if i%2==0:
            a.append(i//2+1)
        else:
            a.append(-(i//2+1))
    
    print(*a)
    
main()
```

---

## 4. 시간/공간 복잡도

* 시간: $O(N)$
* 공간: $O(N)$
