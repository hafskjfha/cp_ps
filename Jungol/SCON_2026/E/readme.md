---

## 1. 문제 요약
- 문제 핵심: 
  - 길이가 $N$인 이진문자열 $S$가 주어진다. 다음 연산을 원하는 만큼 수행해 모든 $1\le i<N$에 대해서 $S_i \neq S_{i+1}$로 만들고 싶다. 필요한 연산의 최소 횟수를 구하여라.
  - 연산: 두 정수 $l,r(1\le l \le r \le N)$을 선택한후 부분 문자열 $S_l S_{l+1} \dots S_{r-1}S_{r}$을 뒤집는다. 즉 `0`은 `1`로 `1`은 `0`으로 바뀐다.
- 입력/출력 조건: 
- 제한 조건: $1\le N \le 2\times 10^5$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 같은 것 끼리 있는 것을 압축하여 처리하는 것인가 라고 생각했다.
- 막혔던 포인트:
  - 하지만 반례가 있었고 어떻게 선택하는 것이 최선인지 알지 못하였다.
- 답지 참고 후:
  - 조건에 맞는 목표문자열은 `0101...`과 `1010...`만 있다.
  - 따라서 원본 문자열에서 목표 문자열과 xor한 비트가 1인 부분을 뒤집어 주면 된다.
  - 근데 1인 부분이 블록을 이루는 경우에는 한번에 뒤집을 수 있으므로 정답은 $S$ xor 목표문자열의 1블록 개수중 최소(목표문자열이 2개 이므로)이다.

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py
import sys

def main():
    input=sys.stdin.readline
    n=int(input())
    s=input().strip()
    
    t0,t1=[],[]
    for i in range(n):
        if i%2==0:
            t0.append(+('0'!=s[i]))
            t1.append(+('1'!=s[i]))
        else:
            t0.append(+('1'!=s[i]))
            t1.append(+('0'!=s[i]))
    
    print(min(len([x for x in "".join(map(str,t0)).split("0")if x]),len([x for x in "".join(map(str,t1)).split("0")if x])))
        
    
main()
```

* 시간: $O(N)$
* 공간: $O(N)$
