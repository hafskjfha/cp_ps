# 문제 제목: [사전순 최소](https://jungol.co.kr/contest/4101/problem/4)

## 1. 문제 요약
- 문제 핵심: 
  - 문자열 $T$가 있고 다음 연산을 0회 이상 원하는 만큼 사용하여 만들 수 있는 모든 문자열 중, 사전 순으로 가장 앞서는 문자열을 구하여라. 
  - 연산: 두 정수 $l, r$를 선택한다. $(1 \le l < r \le |T|)$, 만약 부분 문자열 $T_l T_{l+1} \dots T_r$의 모든 문자가 동일하다면, 그 구간에서 문자 하나만 남기고 나머지를 모두 제거한다. 즉, $T_l = T_{l+1} = \cdots = T_r = \large{c}$이라면 $T_1 T_2 \dots T_{l-1}\, \underline{T_l T_{l+1} \dots T_r}\, T_{r+1} \dots T_{|T|} \to  T_1 T_2 \dots T_{l-1}\, \underline{\large{c}}\, T_{r+1} \dots T_{|T|}$이 된다.
- 입력/출력 조건: 
- 제한 조건: $1\le |T| \le 2\times 10^5$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 같은 문자들을 1개로 압축할수 있다는 점과 사전순으로 가장 앞서는 문자열로 만들기 위해 관찰했다.
  - 만약 특정 문자 $T_i$에 대해서 $T_{i+k}$가 같지 않으면서 $T_{i+k}$이 $T_i$보다 사전순으로 앞서다면 $T_i$와 인접하면서 같은 문자들을 압축하는 것이 이득이고 그렇지 않다면 유지하는 것이 이득이라고 생각했다.
  - 같은 문자를 여러 개 유지하면 그만큼 뒤의 작은 문자의 등장 시점이 늦어진다. 사전순에서는 가능한 한 빨리 작은 문자가 등장하는 것이 유리하므로, 뒤에 더 작은 문자가 존재하면 현재 문자의 연속 구간을 하나로 압축하는 것이 항상 이득이다.
