---

## 1. 문제 요약
- 문제 핵심: 
  - 길이 $N$인 수열 $A$에 대해서 $A$의 두 원소 중간에 원하는 수를 삽입하는 연산을 원하는 만큼 수행할 수 있다. 
  - 연산을 수행하여 인접한 두 원소의 차의 절댓값이 1이 되게 하려고 할때 사용한 연산의 최솟값을 구하시오.
- 입력/출력 조건:
- 제한 조건: $2\le N \le 2\times 10^5, 1\le A_i \le 10^9$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 인접한 두 원소의 절댓값이 1이 아니라면 연산을 무조건 수행하되, 만약 인접한 원소가 같다면 연산을 1번 수행하여 조건을 만족시키게 하고 만약 차의 절댓값이 2보다 크다면 차이의 절댓값-1번의 연산을 수행하면 된다고 생각했다.
- 막혔던 포인트:
  - x
- 답지 참고 후:
  - x

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py
import sys

def main():
    input=sys.stdin.readline
    n=int(input())
    a=[*map(int,input().split())]
    
    r=0
    for i in range(1,n):
        if abs(a[i]-a[i-1])!=1:
            if abs(a[i]-a[i-1])==0:r+=1
            else:r+=abs(a[i]-a[i-1])-1
    
    print(r)
    
main()
```

---

## 4. 시간/공간 복잡도

* 시간: $O(N)$
* 공간: $O(1)$
