# 문제 제목: [두 수열](https://doj.kr/ko/problems/342)
- **플랫폼**: DOJ
- **난이도**: 1.6/10
- **풀이 유형**: 구현 / 수학

---

## 1. 문제 요약
- 문제 핵심:
  - 길이가 $N$인 두 수열 $A$와 $B$가 있다. $i,j(1 \le i,j \le N)$를 골라 $A_i \times B_j$를 만들때 만들 수 있는 최댓값을 구하시오.
- 입력/출력 조건: 
- 제한 조건: $1\le N \le 2 \times 10^5, -10000 \le A_i,B_i \le 10000$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - A,B의 최댓값이 양수일때는 정답이 최댓값\*최댓값 또는 최솟값\*최솟값(둘다 음수이면서 절댓값이 최댓값보다 큰 경우가 있으니)이라고 생각했다.
  - 아닌 경우는 무조건 음수만 나오는 경우는 0이 있으니 곱해 줄 수 있다고 생각했고 아니면 최솟값*최댓값을 확인해 최댓값을 구해주었다.
- 막혔던 포인트:
  - x
- 답지 참고 후:
  - $\max(\min(A)\min(B),\min(A)\max(B),\max(A)\min(B),\max(A)\max(B))$

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py
import sys

def main():
    input=sys.stdin.readline
    n=int(input())
    a=sorted(map(int,input().split()))
    b=sorted(map(int,input().split()))
    
    print(max(
        a[0]*b[0],
        a[-1]*b[0],
        a[0]*b[-1],
        a[-1]*b[-1]
    ))
    
    
main()
```

---

## 4. 시간/공간 복잡도

* 시간: $O(N)$
* 공간: $O(1)$
