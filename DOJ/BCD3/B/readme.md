- **플랫폼**: DOJ

- **풀이 유형**: 해구성

---

## 1. 문제 요약
- 문제 핵심: 
  - `6`과 `7`로만 구성된 길이가 $N$인 10진법 수 $S$가 주어진다.
  - $S$에서 최대 $\lfloor \frac{N}{2} \rfloor$문자를 지워 6의 배수또는 7의 배수를 만들고 싶을떄 조건을 만족하는 결과물 중 하나를 출력하라.
- 입력/출력 조건: 
- 제한 조건: $1\le N \le 3\times 10^5$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 문자를 최대 절반을 지울수 있으므로 6의 등장 개수가 절반보다 적다면 6을 모두 지워 7의 배수를 만들고 7의 등장 개수가 절반보다 적다면 7을 모두 지워 6의 배수를 만들면된다.
- 막혔던 포인트:
  - x
- 답지 참고 후:
  - x

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py
def main():
    input=open(0).readline
    n=int(input())
    st=input()
    
    if st.count('6')<=n//2:
        print('7'*st.count('7'))
    else:
        print('6'*st.count('6'))
        


main()
```

---

## 4. 시간/공간 복잡도

* 시간: $O(N)$
* 공간: $O(N)$

---

## 5. 배운 점 / 실수

* 6의 배수만드는법, 7의 배수 만드는 법을 생각하느라 30분넘게 삽질을 해버렸다.
* 결과물의 조건에 집중하는것이 아니라 주어진 제약(연산의 사용횟수 같은거)에 대해서 먼저 생각해봐야 함을 알게되었다.
