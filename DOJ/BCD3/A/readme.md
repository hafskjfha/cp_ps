# 문제 제목: [67 sequence 1](https://dojoi.xyz/ko/problems/153)
- **플랫폼**: DOJ
- **난이도**: 1/10
- **풀이 유형**: 구현

---

## 1. 문제 요약
- 문제 핵심: 
  - $0,1,\dots,9$로만 구성된 길이가 $N$인 문자열 $S$에서 문자를 하나 지우는 연산을 할수 있다.
  - 연산을 사용하여 문자열에 6과 7이 없게 하고싶을때 사용하는 최소연산수를 구하라.
- 입력/출력 조건: 
- 제한 조건: $1 \le N \le 3\times 10^5$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 6과 7만 지우면 최적이므로 6과 7가 등장한 개수를 출력하면 된다.
- 막혔던 포인트:
  - x
- 답지 참고 후:
  - x

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py
def main():
    input=open(0).readline
    input()
    s=input().strip()
    print(s.count('6')+s.count('7'))
    

main()
```

---

## 4. 시간/공간 복잡도

* 시간: $O(N)$
* 공간: $O(1)$
