## 1. 문제 요약
- 문제 핵심: 
  - 세과녁 $A,B,C$가 있고 각 과녁에는 $N$개의 숫자가 적혀있으며 모든 숫자는 단 한번씩($A$에 있는 수는 $B,C$에 등장하지도 않고 $A$에서도 2개이상이 아님)있다.
  - 각 과녁에서 각 구역에 화살이 맞을 확률은 $\frac{1}{N}$일때 세개의 과녁에 한발씩 화살을 쏜다.
  - 화살이 맞은 세 구역에 적힌 세 수 중 가장 큰 값이 적힌 과녁이 $A$일 확률, $B$일 확률, $C$일 확률을 각각 구하여라.
- 입력/출력 조건: 각 정답의 확률 $p$에 대해 $p=\frac{x}{y}$이며 $y \not\equiv 0 \pmod{998,244,353}$일때 $x \times y^{-1} \pmod{998,244,353}$ 을 출력하라.
- 제한 조건: $1\le N \le 3\times 10^5, 1\le A_i,B_i,C_i\le 10^9$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 각 과녁의 원소에 대해 돌때 그것이 해당 과녁판의 최대원소이면서 다른 과녁의 원소들 보다 높을 경우를 확률로 계산하면 될 것 같았다.
- 막혔던 포인트:
  - $O(N^2)$풀이 밖에 떠오르지 않았다.
- 답지 참고 후:
  - 모든 과녁을 오름차순으로 정렬한다.
  - 이후 $X$과녁에서 $k$를 맞췄을때 $k$가 최대일 확률은 $Y$에서 $k$보다 작은 원소수 * $Z$에서 $k$보다 작은 원소수 이다. 과녁의 원소들은 오름차순으로 정렬되어 있으므로 이분탐색으로 $k$보다 작은 원소수를 $O(\log N)$으로 구할 수 있다.
  - 따라서 $X$가 가장 큰 값이 적힌 과녁이 될 확률은 $\sum_{i=1}^{N} (Y\text{에서}X_i보다 작은 수의 개수) \times (Z\text{에서}X_i보다 작은 수의 개수)$을 전체 경우의 수인 $N^3$으로 나눈것이고 이를 출력형식에 맞게 출력하면 된다.

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py
from bisect import bisect_left

def main():
    input=open(0).readline
    MOD=998244353
    n=int(input())
    t=[sorted(map(int,input().split()))for _ in range(3)]
    
    for i in range(3):
        ts=0
        for x in t[i]:
            ts+=bisect_left(t[[1,2,0][i]],x)*bisect_left(t[[2,0,1][i]],x)
        print((ts*pow(pow(n,3,MOD),-1,MOD))%MOD,end=' ')


main()
```

---

## 4. 시간/공간 복잡도

* 시간: $O(N\log N)$
* 공간: $O(N)$

---

## 5. 배운 점 / 실수

* 특정수보다 작은 수의 개수를 찾는건 정렬과 이분탐색을 통해 최적화 할 수 있음을 알았다.
* 확률 문제라도 직접 확률을 다루기보다, 가능한 경우의 수를 세고 전체 경우의 수로 나누면 조합/탐색 문제로 바뀐다는 점을 알게되었다.
