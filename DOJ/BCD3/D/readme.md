# 문제 제목: [빨강과 파랑](https://dojoi.xyz/ko/problems/156)
- **플랫폼**: DOJ
- **난이도**: 3.3/10
- **풀이 유형**: 누적합

---

## 1. 문제 요약
- 문제 핵심: (짧게 2~3문장)
  - `R`또는 `B`로만 구성된 길이 $N$인 문자열 $S$가 있고 다음 연산을 수행할 수 있다.
  - $S$의 $i$번째 문자가 `R`이면 `B`로 `B`라면 `R`로 바꾼다.
  - 문자열에서 `R`이 등장하는 모든 위치가 비어 있거나 하나의 연속한 구간을 이루고, `B`가 등장하는 모든 위치도 비어 있거나 하나의 연속한 구간을 이루게 만들고 싶다. 필요한 연산 횟수의 최솟값을 구하여라.
- 입력/출력 조건: 
- 제한 조건: $1\le N \le 3\times 10^5$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - $i$번째 구간을 기점으로 왼쪽은 R, 오른쪽은 B로 하거나 왼쪽을 B, 오른쪽을 R로 해야함을 알았다.
- 막혔던 포인트:
  - $O(N^2)$풀이 밖에 떠오르지 않았다.
- 답지 참고 후:
  - 누적합으로 `B`가 등장한 횟수를 관리하여 최적화를 한다.

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py
def main():
    input=open(0).readline
    n=int(input())
    s=input().strip()
    
    ps=[0]*(n+1)
    
    for i in range(n):
        ps[i+1]=ps[i]+(s[i]=='B')
    
    ans=min(ps[-1],n-ps[-1])
    
    for i in range(n):
        ans=min(ans,(i+1)-ps[i+1]+(ps[n]-ps[i+1]),ps[i+1]+(n-i-1)-(ps[n]-ps[i+1]))
    print(ans)


main()
```

---

## 4. 시간/공간 복잡도

* 시간: $O(N)$
* 공간: $O(N)$

---

## 5. 배운 점 / 실수

* 두 문자 문자열에서 “각 문자의 등장 위치가 하나의 연속 구간”이어야 한다면, 최종 문자열은 색이 최대 한 번만 바뀌는 형태다. 따라서 경계 하나를 정하고, 왼쪽/오른쪽에서 바꿔야 하는 문자 수를 누적합으로 빠르게 계산하면 된다.
