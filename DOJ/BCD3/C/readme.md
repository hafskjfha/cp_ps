## 1. 문제 요약
- 문제 핵심: 
  - 길이가 $N$이고 $0,1,\dots,9$로만 구성된 문자열 $S$가 있고 한 번의 연산으로 현재 문자열에서 서로 이웃한 두 문자를 골라 동시에 지울 수 있다. 문자를 지운 뒤에는 남은 문자들이 순서를 유지한 채 서로 붙는다.
  - 당신은 최소 횟수의 연산을 사용하여, 남은 문자열에 문자 6과 문자 7이 한 번도 등장하지 않게할때 필요한 연산 횟수의 최솟값을 구하시오.
- 입력/출력 조건: 불가능하다면 -1을 출력
- 제한 조건: $1\le N \le 3\times 10^5$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 문제를 잘못 읽어서 6과 7끼리만 지울수 있다고 생각해서 삽질을 했다.
  - 6또는 7옆에 아무문자가 있다면 그것을 선택하되 이웃한 6또는 7을 선택하는게 이득이라고 생각했다.
- 막혔던 포인트:
  - 아이디어의 구현과 정당성을 보이는데 막혔다. 
- 답지 참고 후:
  - 현재 남은 문자열에서 가장 왼쪽의 6 또는 7의 위치를 $x$라 하자. $x$가 마지막 위치가 아니라면, 이 문자는 $[x, x+1]$을 지워 처리하는 최적해가 존재한다. $x$ 왼쪽에는 더 이상 제거해야 할 문자가 없으므로 왼쪽 문자를 같이 지우는 것은 이득이 없고, 오른쪽 문자를 같이 지우는 것이 손해가 되지 않는다.
  - 다만 오른쪽 끝에 있는 것을 예외처리 해주어야 한다.
  - 그렇다면 연속된 6 or 7의 길이를 잡고 그 구간길이 $L$에 대해서 $\lceil \frac{L}{2} \rceil$을 하면 그 구간에 있는 6또는 7을 모두 지울수 있다. ceil하는 이유는 67구간의 길이가 홀수라면 다른 수 한개를 가져와서 지워야 하기때문이다. 따라서 그값들을 모두 더하면 답이다.
  - 한 번의 연산마다 문자 2개를 지우므로 전체 문자열에서 수행 가능한 연산 횟수는 최대 $\lfloor \frac{N}{2} \rfloor$번이다. 따라서 필요한 연산 수가 이를 넘으면 불가능하다.

## 3. 코드 정리
[정답 코드](./answer.py)
```py
import sys,math

def main():
    input=sys.stdin.readline
    n=int(input())
    s=input().strip()
    
    ans=0
    j=-1
    for i in range(n):
        if s[i] in "67":
            if j==-1:j=i
        else:
            if j!=-1:
                ans+=math.ceil((i-j)/2)
                j=-1
    
    if j!=-1:
        ans+=math.ceil((n-j)/2)
    
    print(ans if ans<=n//2 else -1)


main()
```
