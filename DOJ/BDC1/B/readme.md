## 1. 문제 요약
- 문제 핵심:
  - 길이가 $N$인 정수 수열 $A$가 있을때 $A$를 다음연산을 원하는 만큼 사용하여 오름차순 $(A_1 < A_2 < \cdots < A_N)$을 만드려고 할때 연산을 최소한으로 사용하는 횟수는?
  - 연산: $1 \le i \le N$인 $i$를 선택하여 $A_i$를 $A_i + K$으로 바꾼다. $K <0$인 경우에 유의하라.
- 입력/출력 조건: 오름차순을 만족하게 만들기 위해 필요한 연산 횟수의 최솟값을 출력한다.
- 제한 조건: $1\le N \le 2\times 10^5, 1 \le |K| \le 10^9, 0 \le |A_i| \le 10^9 (1\le i \le N)$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - $K$가 양수이면 $A_1 < A_2 < \cdots < A_N$ 을 만족시키기 위해 $A_2$부터 $A_N$까지 돌면서 $A_i$에 $\max(0,\lfloor \frac{A_{i-1}-A_i+1}{K} \rfloor)$만큼 연산을 사용하여 $A_i$를 업데이트 하면 됩니다.
  - $K$가 음수이면 오름차순을 만족시키기 위해 $A_{N-1}$부터 $A_1$까지 돌면서 $A_i$에  $\max(0,\lfloor \frac{A_{i}-A_{i+1}+1}{|K|} \rfloor)$만큼 연산을 사용하여 $A_i$를 업데이트 하면 됩니다.
  - 그리고 $K$와 $a_i$의 절댓값이 크므로 실수연산을 정수 연산으로 바꾸었습니다.
- 막혔던 포인트:
  - x
- 답지 참고 후:
  - x

## 3. 코드 정리
[정답 코드](./answer.py)
```py
def main():
    input=open(0).readline
    n,k=map(int,input().split())
    a=[*map(int,input().split())]
    r=0
    
    if k>0:
        for i in range(1,n):
            t=max(0,(a[i-1]-a[i]+1)//k+(((a[i-1]-a[i]+1)%k)!=0))
            r+=t
            a[i]+=k*t
            
    else:
        tk=-k
        for i in range(n-2,-1,-1):
            t=max(0,(a[i]-a[i+1]+1)//tk+(((a[i]-a[i+1]+1)%tk)!=0))
            r+=t
            a[i]+=k*t
            
    print(r)
        
        

main()
```

## 4. 시간/공간 복잡도

* 시간: $O(N)$
* 공간: $O(N)$

## 5. 배운 점 / 실수

* 문제에서 K<0에 대해 강조하지 않았어라도 풀 수 있게 문제를 꼼꼼히 읽어야 겠음을 느꼈습니다.
* ceil연산을 정수 연산으로 더 깔끔하게 하는 식 $\lceil \frac{a}{b} \rceil = \lfloor \frac{a+b-1}{b} \rfloor$을 다시 외워가게 되었습니다.
