# 문제 제목: [이진 수열 만들기](https://dojoi.xyz/ko/problems/162)
- **플랫폼**: DOJ
- **난이도**: 2.7/10
- **풀이 유형**: 구현 / 많은 조건 분기

---

## 1. 문제 요약
- 문제 핵심:
  - 이진 수열에 대해 다음 조건을 만족하는 문자열이 존재한다면 출력하시오.
  - 부분문자열 `01`이 $a$번, `10`이 $b$번, `11`이 $c$번, `00`이 $d$번 존재해야 한다.
- 입력/출력 조건: 조건을 만족하는 문자열이 없다면 -1을 출력한다.
- 제한 조건: $0\le a,b,c,d, a+b+c+d\le 2 \times 10^5$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 먼저 $c$과 $d$의 값와 문자열 연관성을 생각했다. 하지만 그렇게 생각하니 예외케이스가 너무 많아져 생각을 다시했다.
  - $a$과 $b$의 개수에 관련이 있다.
    - $a = b$일때
      - $c>0$이면서 $d>0$이고 $a=b=0$이면 조건을 만족하는 문자열은 없다.
      - $c>0, d=0$또는 $c=0, d>0$이면서 $a=b=0$이면 가능하다. 해당 문자로 나열만 하면 되니까.
      - $a>0$ 이면 `011...1101010100...` 으로 조건을 만족하는 문자열을 구성가능하다.
    - $a \neq b$일때
      - $|a-b|>1$이면 조건을 만족하는 문자열은 없다. 왜냐하면 0으로 시작한경우 01/10/01이 번갈아 가면서 밖에 나올수 없고 1로 시작한 경우에도 마찬가지 이므로 2개이상 차이가 날 수 없다.
      - $a>b$이면 `00...00101..010111...`으로 문자열을 구성할 수 있다.
      - $b>a$이면 `11...10101..101000...`으로 문자열을 구성할 수 있다.
- 막혔던 포인트:
  - x
- 답지 참고 후:
  - x

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py
def main():
    input=open(0).readline
    a,b,c,d=map(int,input().split())
    if a==b:
        if a==0:
            if c>0 and d>0:
                print(-1)
            else:
                if c>0:print("1"*(c+1))
                else:print("0"*(d+1))
        else:
            print("01"+"1"*c+"01"*(a-1)+"0"*(d+1))
            
    else:
        if abs(a-b)>1:
            print(-1)
        else:
            if a>b:
                print("0"*d+"01"*a+"1"*c)
            else:
                print("1"*c+"10"*b+"0"*d)
            
        

main()
```

## 4. 시간/공간 복잡도

* 시간: $O(a+b+c+d)$
* 공간: $O(a+b+c+d)$
