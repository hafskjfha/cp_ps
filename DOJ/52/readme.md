## 1. 문제 요약
- 문제 핵심: 
  - 길이가 $N$인 문자열 $S$가 있다. 다음 연산을 최대 $N$번 사용하여 만들 수 있는 부분 문자열 `GSHS`의 최대 개수를 출력한다. 
  - 연산: $1\le l \le r \le N$인 $(l,r)$을 고른후 $S_l,S_{l+1},\dots,S_{r-1},S_r$을 $S_r,S_{r-1},\dots,S_{l+1},S_l$로 바꾼다.
- 입력/출력 조건: 
- 제한 조건: $1\le N \le 10^6$

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - 문자열을 어떻게 회전해야 최적일까 생각하려고 했다.
- 막혔던 포인트:
  - 회전연산과 가능한 횟수 $N$의 관계를 파악하지 못하였다.
- 답지 참고 후:
  - $S_i$에 $S_j$를 가져오고 싶으면 $(i,j)$을 선택하여 회전을 수행하면 위치를 바꿀 수 있다. 
  - 따라서 연산을 반복적으로 사용한다면 원하는 문자열로 재배치가 가능하다. 문자열 길이가 $N$일 때 선택 정렬처럼 최대 $N-1$번의 교환으로 원하는 배열을 만들 수 있으므로, 허용된 연산 횟수 $N$번 안에 임의의 재배열이 가능하다.
  - 따라서 `GSHS`의 최대 개수는 $C_k$를 $S$에서 문자 $k$의 개수라고 했을때 $\min(C_G,\lfloor \frac{C_S}{2} \rfloor, C_H)$이다.

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py
def main():
    input=open(0).readline
    n=int(input())
    s=input().strip()
    gc,sc,hc=0,0,0
    for c in s:
        if c=="G":gc+=1
        elif c=="S":sc+=1
        elif c=="H":hc+=1
    
    print(min(gc,sc//2,hc))
    
    
main()
```

* 시간: $O(N)$
* 공간: $O(N)$
