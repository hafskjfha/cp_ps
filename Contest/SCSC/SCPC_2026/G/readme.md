# 문제 제목: [SCSC/SCPC_2026/div3/G SCSC게임](https://atcoder.jp/contests/scpc2026-div3/tasks/scpc2026_div3_g)
- **플랫폼**: 앳코더
- **난이도**: 7.1/10
- **풀이 유형**: 게임 이론 / 홀짝성

---

## 1. 문제 요약
- 문제 핵심: 
  - `S`, `C`로만 구성된 문자열 $S$가 주어진다.
  - P1과 P2가 서로 번갈아가며 $S$의 문자 한개를 지운다.
  - 문자 한개를 지워 $S$의 부분문자열에 `SCSC`이 있다면 해당 플레이어가 승리한다.
  - 만약 지울 수 있는 문자가 없다면 해당 플레이어는 패배한다.
- 입력/출력 조건: 
- 제한 조건: 
  - $1 \le T \le 10,000$
  - $1\le |S| \le 200,000$
  - $S$는 `S`,`C`로만 구성되어 있으며 처음에는 `SCSC`를 부분문자열로 가지지 않는다.
  - 모든 테스트케이스의 $|S|$의 합은 $200,000$을 넘지 않는다.

---

## 2. 접근 아이디어
- 자력 풀이 시도:
  - `SSCSC`, `SCCSC`와 같은 부분문자열이 있다면 한개의 문자를 지워 `SCSC`를 부분문자열로 만든다면 P1이 이긴다.
  - 만약 해당 부분문자열을 갖지 않는다면 어떤수가 최선일까에 대해서 고민해본다.
- 막혔던 포인트:
  - `SSCCSC` 와 같은 문자열에서 어떻게 처리해야할지 막혔다.
- 답지 참고 후:
  - `SSCSC`, `SCCSC`와 같이 한턴에 이길 수 있는 경우가 아니라면 안전문자열이고 이때 승리로 연결 할 수 있는 수는 없다.
  - 따라서 양끝의 문자를 지우는 것이 최선이 되며 그럼 $|S|$가 홀수 일때는 P1이 짝수라면 P2가 이기는 게임이 된다.

---

## 3. 코드 정리
[정답 코드](./answer.py)
```py
def main():
    input=open(0).readline
    for _ in range(int(input())):
        s=input().strip()
        if len(s)%2==1 or "SSCSC" in s or "SCCSC" in s or "SCSSC" in s or "SCSCC" in s:
            print("Terra")
        else:
            print("Lulu")

main()
```

---

## 4. 시간/공간 복잡도

* 시간: $O(T\times|S|)$
* 공간: $O(|S|)$
